# 🌿 Renovaciclo | Plataforma de Economia Circular

> Projeto acadêmico desenvolvido no Senac Alagoas com foco em sustentabilidade, logística inteligente e impacto social.

## 📌 Sobre o Projeto

A Renovaciclo é uma plataforma B2B voltada para gestão e transformação de resíduos orgânicos e industriais.  
O objetivo é conectar grandes geradores de resíduos a novas cadeias produtivas, reduzindo custos de descarte e gerando valor econômico e ambiental.

Este projeto simula uma solução real aplicada ao contexto de Maceió – AL, com foco em logística eficiente, ESG e economia circular.

---

## 🚀 Principais Funcionalidades

- Estruturação de logística com conceitos de **Just-in-Time** e **Curva ABC**
- Transformação de excedentes alimentares em insumos de alto valor
- Proposta de consultoria ESG para empresas
- Interface moderna com foco em experiência do usuário

---

## 🛠️ Stack e Conceitos Aplicados

- **Vite** (ambiente moderno de build)

- **HTML5 & CSS3** (layout responsivo e design system)
- **AOS Library** (animações baseadas em scroll)
